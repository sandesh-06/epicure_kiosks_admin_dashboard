# Epicure Admin Dashboard  

Welcome to the **Epicure Admin Dashboard** project! This application provides an intuitive interface for managing kiosks, dispensers, and recipes in the Epicure ecosystem.

## Project Structure

The project is organized into two main folders:

- **client/**: Contains the frontend code built with React using Vite.
- **server/**: Contains the basic backend setup and model, ready for future API integrations.

## Setup and Running the Program

To set up and run the application, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/sandesh-06/epicure_kiosks_admin_dashboard.git
   ```

2. **Navigate to the client folder**:
   ```bash
   cd epicure_kiosks_admin_dashboard/client
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the application**:
   ```bash
   npm run dev
   ```

5. **Visit the website**: Alternatively, you can visit the application using this [link](https://epicure-kiosks-admin-dashboard.vercel.app/).

## Local Storage Integration
The **Epicure Admin Dashboard** currently utilizes local storage for fetching and storing items. This simulates API interactions and allows for quick data retrieval and persistence. Using the same logic, APIs can be integrated easily in the future, enabling more dynamic and scalable data management.

## Functionalities

The **Epicure Admin Dashboard** includes the following key functionalities:

- **Kiosks Page**: Displays all available kiosks. 
- **Kiosk Details Page**: Click on a kiosk to view its details, including:
  - **Dispenser Details**: Information about the dispensers associated with the kiosk.
  - **Recipe Details**: Overview of the recipes linked to the kiosk.
  - **Kiosk Status**: Current status of the kiosk.
- **Dispenser Management**: Add and edit dispensers as needed.
- **Recipe Management**: Add and edit recipes in the system.
- **Recipe Details Page**: Click on a recipe to view:
  - Recipe Name
  - Category
  - Ingredient Details: Add or edit ingredients related to the recipe.

## Additional Pages

The dashboard also includes additional pages for better navigation and management:

- **List of Dispensers**: A dedicated page displaying all dispensers.
- **List of Recipes**: A dedicated page showcasing all available recipes.
- **List of Ingredients**: A dedicated page providing a comprehensive list of all ingredients.

## Conclusion

The **Epicure Admin Dashboard** is designed to facilitate efficient management of kiosks, dispensers, and recipes, enhancing the overall user experience.
